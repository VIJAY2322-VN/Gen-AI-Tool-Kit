# Generative AI Toolkit

## Overview
Generative AI Toolkit is a Python-based project that demonstrates the implementation of core Generative AI modules and NLP concepts. This project combines multiple AI functionalities into a single toolkit for learning, experimentation, and real-world applications.

The repository includes modules for:
- AI Chatbot Interaction
- RAG analysis
- Text Generation
- Sentiment Analysis
- Text Summarization
- AI Image Prompt Generation

This project is useful for students, beginners in AI/ML, and developers who want to understand how Generative AI systems are structured and implemented.

---

## Features

### AI Chatbot
Provides automated conversational responses using simple NLP logic.

### Text Generator
Generates AI-style textual content dynamically.

### Sentiment Analysis
Detects positive, negative, or neutral sentiment from user text.

### Text Summarizer
Converts lengthy content into concise summaries.

### Image Prompt Generator
Simulates AI image generation based on text prompts.
